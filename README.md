## Description

ShopNGo is a comprehensive e-commerce mobile application designed to provide a seamless shopping experience for users. This full-stack mobile application offers a complete suite of e-commerce functionalities with both frontend and backend integration. The key features of the application include:

- **User Authentication & Profiles**: Secure user registration, login, and personalized profile management with order history and preferences.

- **Product Catalog & Search**: Extensive product browsing with categories, filters, and powerful search functionality to help users find exactly what they need.

- **Shopping Cart & Wishlist**: Intuitive cart management with add/remove items, quantity adjustments, and wishlist capabilities for saving favorite products.

- **Secure Payment Integration**: Multiple payment options with secure transaction processing and order confirmation.

- **Order Tracking & History**: Real-time order status updates and comprehensive order history for users to track their purchases.

- **Admin Dashboard**: Complete backend management for inventory, orders, user management, and analytics.

The front-end of ShopNGo is developed using **Flutter**, providing a beautiful, responsive cross-platform experience that works seamlessly on both Android and iOS devices with a single codebase.

The back-end is powered by **Firebase**, offering robust cloud infrastructure including:
- **Firebase Authentication** for secure user management
- **Cloud Firestore** for real-time database operations
- **Firebase Storage** for product images and media
- **Cloud Functions** for server-side logic and payment processing

## Installation

To install and run ShopNGo on your device, follow these steps:

### Prerequisites
Make sure you have the **Flutter SDK** (version 3.0 or higher) installed and properly configured in your development environment before proceeding. You will also need a **Firebase project** set up to connect the backend.

For detailed instructions on setting up Flutter, refer to the official documentation:
[Getting Started with Flutter](https://flutter.dev/docs/get-started/install)

### Setup Instructions

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/sasindu26/Shopingo-Mobile-Application.git](https://github.com/sasindu26/Shopingo-Mobile-Application.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd Shopingo-Mobile-Application
    ```

3.  **Install the necessary dependencies:**
    ```bash
    flutter pub get
    ```

4.  **Build and run the application:**
    ```bash
    flutter run
    ```
    (Ensure an emulator is running or a device is connected.)
